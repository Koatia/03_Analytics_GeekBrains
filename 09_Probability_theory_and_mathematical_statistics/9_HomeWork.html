Задание 1

Даны значения величины заработной платы заемщиков банка (zp) и значения их поведенческого кредитного скоринга (ks). Используя математические операции, посчитать коэффициенты линейной регрессии, приняв за X заработную плату (то есть, zp - признак), а за y - значения скорингового балла (то есть, ks - целевая переменная). Произвести расчет как с использованием intercept, так и без.

zp = [35, 45, 190, 200, 40, 70, 54, 150, 120, 110]

ks = [401, 574, 874, 919, 459, 739, 653, 902, 746, 832]

Решение:

Для вычисления коэффициентов линейной регрессии вручную, будем использовать следующие формулы:

  1. Коэффициент наклона (slope):

β1=∑i=1n(xi−xˉ)(yi−yˉ)∑i=1n(xi−xˉ)2\quad \beta_1 = \frac{\sum_{i=1}^{n} (x_i - \bar{x})(y_i - \bar{y})}{\sum_{i=1}^{n} (x_i - \bar{x})^2}β1​=∑i=1n​(xi​−xˉ)2∑i=1n​(xi​−xˉ)(yi​−yˉ​)​

  1. Свободный член (intercept):

β0=yˉ−β1⋅xˉ\quad \beta_0 = \bar{y} - \beta_1 \cdot \bar{x}β0​=yˉ​−β1​⋅xˉ

где xˉ\bar{x}xˉ и yˉ\bar{y}yˉ​ - средние значения xxx и yyy соответственно.

  1. Вычисляем средние значения xˉ\bar{x}xˉ и yˉ\bar{y}yˉ​:

xˉ=1n∑i=1nxi=35+45+190+200+40+70+54+150+120+11010=101410=101.4\bar{x} = \frac{1}{n} \sum_{i=1}^{n} x_i = \frac{35 + 45 + 190 + 200 + 40 + 70 + 54 + 150 + 120 + 110}{10} = \frac{1014}{10} = 101.4xˉ=n1​∑i=1n​xi​=1035+45+190+200+40+70+54+150+120+110​=101014​=101.4

yˉ=1n∑i=1nyi=401+574+874+919+459+739+653+902+746+83210=709910=709.9\bar{y} = \frac{1}{n} \sum_{i=1}^{n} y_i = \frac{401 + 574 + 874 + 919 + 459 + 739 + 653 + 902 + 746 + 832}{10} = \frac{7099}{10} = 709.9yˉ​=n1​∑i=1n​yi​=10401+574+874+919+459+739+653+902+746+832​=107099​=709.9

  1. Вычисляем Коэффициент наклона (slope) β1\beta_1β1​:

β1=∑i=1n(xi−xˉ)(yi−yˉ)∑i=1n(xi−xˉ)2\beta_1 = \frac{\sum_{i=1}^{n} (x_i - \bar{x})(y_i - \bar{y})}{\sum_{i=1}^{n} (x_i - \bar{x})^2}β1​=∑i=1n​(xi​−xˉ)2∑i=1n​(xi​−xˉ)(yi​−yˉ​)​

∑i=1n(xi−xˉ)(yi−yˉ)=(35−101.4)(401−709.9)+(45−101.4)(574−709.9)+…+(110−101.4)(832−709.9)=92917.28\sum_{i=1}^{n} (x_i - \bar{x})(y_i - \bar{y}) = (35-101.4)(401-709.9) + (45-101.4)(574-709.9) + \ldots + (110-101.4)(832-709.9) = 92917.28∑i=1n​(xi​−xˉ)(yi​−yˉ​)=(35−101.4)(401−709.9)+(45−101.4)(574−709.9)+…+(110−101.4)(832−709.9)=92917.28

∑i=1n(xi−xˉ)2=(35−101.4)2+(45−101.4)2+…+(110−101.4)2=34714.6\sum_{i=1}^{n} (x_i - \bar{x})^2 = (35-101.4)^2 + (45-101.4)^2 + \ldots + (110-101.4)^2 = 34714.6∑i=1n​(xi​−xˉ)2=(35−101.4)2+(45−101.4)2+…+(110−101.4)2=34714.6

β1=92917.2834714.6≈2.677\beta_1 = \frac{92917.28}{34714.6} \approx 2.677β1​=34714.692917.28​≈2.677

  1. Вычисляем Свободный член (intercept) β0\beta_0β0​:

β0=709.9−2.677⋅101.4≈709.9−271.394=438.506\beta_0 = 709.9 - 2.677 \cdot 101.4 \approx 709.9 - 271.394 = 438.506β0​=709.9−2.677⋅101.4≈709.9−271.394=438.506

Итоговые коэффициенты линейной регрессии

  1. С интерсептом:
    ks=444.18+2.62⋅zpks = 444.18 + 2.62 \cdot zpks=444.18+2.62⋅zp

    ks=5.89⋅zpks = 5.89 \cdot zpks=5.89⋅zp

Задание 2

Посчитать коэффициент линейной регрессии при заработной плате (zp), используя градиентный спуск (без intercept).

Для вычисления коэффициента линейной регрессии при заработной плате (zpzpzp) с использованием градиентного спуска без интерсепта, нам нужно выполнить следующие шаги:

  1. Определить функцию потерь (в данном случае, это среднеквадратичная ошибка).
    L=12m∑i=1m(y^i−yi)2L = \frac{1}{2m} \sum_{i=1}^{m} (\hat{y}_i - y_i)^2L=2m1​∑i=1m​(y^​i​−yi​)2
    где y^i=β1xi\hat{y}_i = \beta_1 x_iy^​i​=β1​xi​.
  2. Определить градиент функции потерь.
    ∂L∂β1=1m∑i=1m(β1xi−yi)xi\frac{\partial L}{\partial \beta_1} = \frac{1}{m} \sum_{i=1}^{m} (\beta_1 x_i - y_i) x_i∂β1​∂L​=m1​∑i=1m​(β1​xi​−yi​)xi​
  3. Итеративно обновлять коэффициенты, используя градиентный спуск.
    β1:=β1−α∂L∂β1\beta_1 := \beta_1 - \alpha \frac{\partial L}{\partial \beta_1}β1​:=β1​−α∂β1​∂L​
    где α\alphaα — шаг обучения (learning rate).
import numpy as np

# Данные
zp = np.array([35, 45, 190, 200, 40, 70, 54, 150, 120, 110])
ks = np.array([401, 574, 874, 919, 459, 739, 653, 902, 746, 832])

# Параметры градиентного спуска
alpha = 0.0001  # Шаг обучения
n_iterations = 10000  # Количество итераций
m = len(zp)  # Количество наблюдений

# Начальное значение коэффициента
beta_1 = 0

# Градиентный спуск
for _ in range(n_iterations):
    gradient = (1/m) * np.sum((beta_1 * zp - ks) * zp)
    beta_1 -= alpha * gradient

print(f"Коэффициент наклона линейной регрессии: {beta_1}")
print(f"Итоговое уравнение регрессии: y = {beta_1:0.2f} * x")

Коэффициент наклона линейной регрессии: 5.889820420132689
Итоговое уравнение регрессии: y = 5.89 * x

Этот результат совпадает с тем, который был получен с использованием функции линейной регрессии из библиотеки scikit-learn для модели без интерсепта. Это подтверждает корректность проведенных вычислений.

Задание 3

Произвести вычисления как в пункте 2, но с вычислением intercept. Учесть, что изменение коэффициентов должно производиться на каждом шаге одновременно (то есть изменение одного коэффициента не должно влиять на изменение другого во время одной итерации).

Для вычисления коэффициента линейной регрессии при заработной плате (zpzpzp) с использованием градиентного спуска без интерсепта, нам нужно выполнить следующие шаги:

  1. Определить функцию потерь (в данном случае, это среднеквадратичная ошибка).
    L=12m∑i=1m(y^i−yi)2L = \frac{1}{2m} \sum_{i=1}^{m} (\hat{y}_i - y_i)^2L=2m1​∑i=1m​(y^​i​−yi​)2
    где y^i=β0+β1xi\hat{y}_i = \beta_0 + \beta_1 x_iy^​i​=β0​+β1​xi​.

  2. Определить градиенты функции потерь.
    ∂L∂β0=1m∑i=1m(β0+β1xi−yi);∂L∂β1=1m∑i=1m(β0+β1xi−yi)xi\frac{\partial L}{\partial \beta_0} = \frac{1}{m} \sum_{i=1}^{m} (\beta_0 + \beta_1 x_i - y_i);\quad \frac{\partial L}{\partial \beta_1} = \frac{1}{m} \sum_{i=1}^{m} (\beta_0 + \beta_1 x_i - y_i) x_i∂β0​∂L​=m1​∑i=1m​(β0​+β1​xi​−yi​);∂β1​∂L​=m1​∑i=1m​(β0​+β1​xi​−yi​)xi​

  3. Итеративно обновлять коэффициенты, используя градиентный спуск.
    β0:=β0−α∂L∂β0;β1:=β1−α∂L∂β1\beta_0 := \beta_0 - \alpha \frac{\partial L}{\partial \beta_0};\quad \beta_1 := \beta_1 - \alpha \frac{\partial L}{\partial \beta_1}β0​:=β0​−α∂β0​∂L​;β1​:=β1​−α∂β1​∂L​
    где α\alphaα — шаг обучения (learning rate).

import numpy as np

# Данные
zp = np.array([35, 45, 190, 200, 40, 70, 54, 150, 120, 110])
ks = np.array([401, 574, 874, 919, 459, 739, 653, 902, 746, 832])

# Параметры градиентного спуска
alpha = 0.0001  # Шаг обучения
n_iterations = 1000000  # Количество итераций
m = len(zp)  # Количество наблюдений

# Начальные значения коэффициентов
beta_0 = 0
beta_1 = 0

# Градиентный спуск
for _ in range(n_iterations):
    y_pred = beta_0 + beta_1 * zp
    error = y_pred - ks
    
    beta_0_gradient = (1/m) * np.sum(error)
    beta_1_gradient = (1/m) * np.sum(error * zp)
    
    beta_0 -= alpha * beta_0_gradient
    beta_1 -= alpha * beta_1_gradient

print(f"Коэффициенты линейной регрессии: {beta_0}, {beta_1}")
print(f"Итоговое уравнение регрессии: y = {beta_0:0.2f} + {beta_1:0.2f} * x")

Итоговое уравнение регрессии: y = 444.18 + 2.62 * x
